# Default Packing List

Minimalist baseline. Copy and adapt per trip — add destination-specific items, remove what doesn't apply.

## Clothes (~3 sets, sport-oriented)

Wash-as-you-go: shower with clothes using shampoo (works as detergent), or bring laundry detergent pods. Hang to dry overnight → clean and dry by next day. This is why 3 sets is enough.

- [ ] 3 sport/moisture-wicking shirts (light, quick-dry, double as daily wear)
- [ ] 1 casual t-shirt or nice shirt (for dinners/outings)
- [ ] 2 pairs shorts or pants (depending on climate)
- [ ] 3 pairs underwear
- [ ] 3 pairs socks
- [ ] Running shoes + running gear
- [ ] Flip flops / sandals
- [ ] Swimwear
- [ ] Light jacket or hoodie
- [ ] Laundry detergent pods (optional — shampoo works too)

## Documents & Money
- [ ] Passport
- [ ] Travel insurance info (saved offline)
- [ ] Booking confirmations (saved offline)
- [ ] Cash / travel card (Revolut)
- [ ] Pen (customs/immigration forms)

## Tech
- [ ] Phone + charger
- [ ] Charging cables (USB-C etc.)
- [ ] Power bank
- [ ] Headphones
- [ ] Kindle
- [ ] Whoop + charger
- [ ] Blue light glasses
- [ ] Power adapter (if needed for destination)
- [ ] Universal power adapter (optional — one for every trip)

## Toiletries
- [ ] Toothbrush + toothpaste
- [ ] Deodorant
- [ ] Sunscreen (face + body)
- [ ] Comb
- [ ] Beard cream
- [ ] Nail trimmer
- [ ] Fluticasone nasal spray

## Supplements (if space allows)
- [ ] Creatine
- [ ] Omega-3

## Travel Comfort
- [ ] Earplugs + eye mask
- [ ] Neck pillow
- [ ] Reusable water bottle
- [ ] Snacks for transit (protein bars, nuts)

## Nice to Have
- [ ] Small padlock (hostel lockers)
- [ ] Ziplock bags (rain/splash protection)
- [ ] Hat/cap
- [ ] Sunglasses
- [ ] Basic meds: ibuprofen, anti-diarrhea
- [ ] Microfiber towel (optional — hostels don't always provide)

## Pre-Departure
- [ ] Charge all devices
- [ ] Take out trash
- [ ] Empty perishables from fridge
- [ ] Save offline maps for destination
- [ ] Download entertainment (movies/podcasts/music offline)
- [ ] Check in online (24-48h before flight)
